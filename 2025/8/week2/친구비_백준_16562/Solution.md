# 친구비

## 문제 설명
- 문제 링크: [링크](https://www.acmicpc.net/problem/16562)
- 핵심 사항:
  - "친구의 친구는 친구다"
  - "가장 적은 비용으로 모든 사람과 친구가 되는 방법을 구하라."
---

## 접근 방법
- "친구의 친구는 친구다" 를 이용하기 위해 분리 집합으로 집합을 나눔
- 각 집합의 최소 친구비를 탐색
  - 각 집합의 루트의 최소 친구비를 별도의 배열에 저장하며, union 연산시에 두개의 집합중 최소 친구비를 새로운 집합 루트의 최소 친구비로 지정
- 모든 집합의 최소 친구비를 합산
